# Human Relationship Principles Website

데일 카네기 핵심 원칙을 기반으로 만든 단일 페이지 웹사이트입니다.

## Included Content
- 인간을 다루는 3가지 기본 원칙
- 사람들이 당신을 좋아하게 만드는 6가지 방법
- 사람을 설득하는 12가지 방법
- 리더의 9가지 원칙
- 핵심 메시지: 인간은 감정으로 움직인다
- 한국어/영어 전환 페이지 (`index.html`, `en.html`)
